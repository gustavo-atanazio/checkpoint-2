# Checkpoint 2

## Sobre o projeto

Aplicativo mobile desenvolvido com Expo e TypeScript que consome uma API para listar usuários e posts, exibir detalhes e permitir a criação de novos posts. O app organiza as telas de listagem e detalhe, utiliza cache para melhorar a experiência e inclui tratamento de carregamento e erros. O objetivo é praticar integração com serviços, navegação e boas práticas de componentes reutilizáveis.

O projeto foi estruturado com telas específicas para usuários, posts e detalhes, além de hooks para centralizar a lógica de dados. Os serviços de API e cache ficam isolados em camadas próprias para facilitar manutenção e testes. Também há componentes de UI reutilizáveis, como cartões de usuário/post e estados de vazio, carregamento e erro, garantindo uma experiência consistente.

### Funcionalidades principais

- Listagem de usuários e posts com estado de carregamento.
- Visualização de detalhes do usuário e seus posts.
- Criação de posts com feedback de sucesso/erro.
- Tratamento de falhas de rede com mensagens amigáveis.
- Reuso de componentes e organização por módulos.
